# Nutrition Coaching Basics

- Prioritize whole foods: lean proteins, vegetables, fruits, whole grains, and healthy fats.
- A practical fat-loss target is a 300-500 kcal daily deficit.
- A practical muscle-gain target is a 150-300 kcal daily surplus.
- Protein target for active adults: around 1.6-2.2 g per kg body weight daily.
- Typical meal structure: protein source + fiber-rich carbs + vegetables + hydration.
- For food image estimation, report ranges and uncertainty when portions are unclear.
- Avoid making medical diagnoses. Refer users to licensed professionals for medical nutrition therapy.
- For sustainability, focus on consistency and weekly trends instead of one meal perfection.
