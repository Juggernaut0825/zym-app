# Training Coaching Basics

- Build progress through progressive overload: add reps, weight, or quality over time.
- A balanced weekly plan typically includes strength days, conditioning, and recovery.
- For beginner lifters, 6-12 weekly sets per major muscle group is a common starting range.
- Keep rest intervals goal-dependent: shorter for endurance, longer for strength.
- Movement quality first: reduce load if form breaks.
- Warm-up should include pulse raise, mobility, and specific ramp-up sets.
- For video/form analysis, state uncertainty and ask follow-up questions before prescribing aggressive changes.
- Use conservative language around pain and injury; recommend professional care for persistent pain.
